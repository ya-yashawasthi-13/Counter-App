# Counter App

A simple, elegant web-based counter application with a modern dark theme UI.

## Features

- **Increment**: Increase the counter by 1
- **Decrement**: Decrease the counter by 1
- **Reset**: Reset the counter back to 0
- **Negative Number Indicator**: The counter display changes to red when the value goes below 0
- **Responsive Design**: Clean, centered card-based layout with smooth animations
- **Interactive Buttons**: Hover and click animations for better user experience

## How to Use

1. Open `index.html` in your web browser
2. Click the buttons to interact with the counter:
   - **+ 1** (Green button): Increment the counter
   - **− 1** (Red button): Decrement the counter
   - **Reset** (Gray button): Reset counter to 0
3. The counter value is displayed in large text in the center of the card

## Styling

- **Dark Theme**: Dark background with modern color scheme
- **Color Coded Buttons**:
  - Green (`#4ecca3`) for increment
  - Red (`#ff6b6b`) for decrement
  - Gray for reset
- **Visual Feedback**: Buttons scale and change opacity on hover/click
- **Negative Number Styling**: Counter text turns red when the value is negative

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
